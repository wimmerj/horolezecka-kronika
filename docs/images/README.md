## Optimalizace obrázků

Před nahráním doporučujeme optimalizovat obrázky:

### Online nástroje
- [TinyPNG](https://tinypng.com/) - komprese JPG a PNG
- [Squoosh](https://squoosh.app/) - pokročilá optimalizace

### Desktop nástroje
- **GIMP** - zdarma, pro Windows/Mac/Linux
- **Adobe Photoshop** - profesionální nástroj

### Nástroje pro odstranění pozadí (PNG s průhledností)
- [Remove.bg](https://www.remove.bg/) - automatické odstranění pozadí online (zdarma)
- [PhotoRoom](https://www.photoroom.com/) - AI odstranění pozadí
- **GIMP** - manuální odstranění pozadí (Select by Color → Delete)
- **Photoshop** - Magic Wand / Quick Selection Tool

### Doporučená kvalita
- JPG kvalita: 85% (dobrá rovnováha mezi kvalitou a velikostí)
- PNG kvalita: 100% (pro průhlednost)
- Maximální velikost souboru: 500 KB pro galerie, 200 KB pro thumbnaily, 300 KB pro PNG profilové fotky

1. Optimalizujte obrázek (viz výše)
2. Uložte do správné složky (podle roku)

